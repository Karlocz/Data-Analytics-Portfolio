# Data Analytics Portfolio

Welcome to my Data Analytics Portfolio.

This repository contains a collection of projects where I analyze real-world datasets to extract insights, identify patterns, and support data-driven decision making.

The goal of these projects is to demonstrate practical skills in the full data analysis workflow:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Business insights and recommendations

All analyses are performed using widely used tools in the data industry.
